## 🚀 How to Run

Open `index.html` in your browser. That is the website entry point.

## 📌 Notes on the Current Setup

- `index.html` remains in the project root.
- `css/` holds styling files.
- `js/` holds JavaScript files.
- `assets/images/` holds image assets.
- `README.md` explains the structure.

## 📝 Correct Linking

From `index.html`, use these paths:

```html
<link rel="stylesheet" href="css/styles.css" />
<script src="js/script.js"></script>
<img src="assets/images/web icon.jpg" alt="Web Development" />
```

## ✅ Recommended Structure

This is the current correct structure for your project:

- `index.html` — main web page
- `css/styles.css` — styles and layout
- `js/script.js` — interactive behavior
- `assets/images/` — image assets used on the site
- `README.md` — project documentation
